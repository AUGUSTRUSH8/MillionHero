# million-hero
百万英雄，自动搜题程序

**最近全民答题比较火爆，比如百万英雄，冲顶大会，芝士超人等，让普通民众也感受了一把心跳。**

> 本程序参考了一下，最近比较火的跳一跳的原理
1. 安卓手机调试模式
2. 截图
3. 截取文字部分
4. ORC文字识别
5. 调用百度，返回查询结果
